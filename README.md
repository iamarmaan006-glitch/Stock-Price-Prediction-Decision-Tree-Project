📈 Stock Price Prediction using Decision Tree
📌 Overview

This project is a Machine Learning web application that predicts stock price movement (UP 📈 or DOWN 📉) using a Decision Tree Classifier based on technical indicators derived from stock data.

🚀 Features
Synthetic stock dataset generation
Technical indicators calculation:
Moving Averages (MA5, MA20, MA50)
RSI (Relative Strength Index)
MACD
Bollinger Bands
Volatility & Returns
Decision Tree Classifier model
Model tuning (depth, split, leaf, criterion)
Model evaluation:
Accuracy score
Confusion matrix
Classification report
Feature importance visualization
Decision tree visualization
Real-time prediction with probability
Download predictions as CSV
🛠️ Tech Stack
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Streamlit
